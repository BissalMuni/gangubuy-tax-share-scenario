# Project: share-scenario

지방세법령 개정 세금정보 유튜브 영상 시나리오 제작·공유·협업 플랫폼.

- Use pnpm as package manager
- Use vitest for testing
- Code comments in Korean
- Commit messages in English
- Follow spec-kit workflow: constitution → spec → plan → implement
- Refer to `.spec/` for project specifications
- Database: Supabase (PostgreSQL + Auth + Storage)
- Deploy: Vercel
- Feature-based folder structure (`src/features/`)
- Server Components by default, Client Components only when needed
- Tree structure defined by JSON config, not folder structure
